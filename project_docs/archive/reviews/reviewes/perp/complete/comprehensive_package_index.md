## üîÑ VANNA AI FRAMEWORK OVERVIEW

### What Vanna Does

Vanna is a **RAG-based natural language-to-SQL framework** that:

‚úÖ **Converts natural language to SQL**
- "What are the top 10 customers?" ‚Üí SQL query
- Supports complex queries with joins & aggregations

‚úÖ **Works with any database**
- PostgreSQL, MySQL, Oracle, SQLite, Snowflake, BigQuery, etc.

‚úÖ **Works with any LLM**
- OpenAI, Claude, Gemini, local Ollama, AWS Bedrock, etc.

‚úÖ **Learns from usage**
- Improves accuracy over time
- Stores successful queries for future reference

‚úÖ **Keeps data private**
- Database contents never sent to LLM
- Execution happens locally

### What Vanna Does NOT Do

‚ùå ETL or data transformation  
‚ùå Report generation  
‚ùå Real-time monitoring  
‚ùå Data security/access control  
‚ùå Complex ML predictions  
‚ùå Data modification (writes)  

### Vanna Architecture

```
User Question
    ‚Üì
[Retrieve Training Data - RAG] ‚Üê Vector Store
    ‚Üì
[Generate SQL - LLM Provider]
    ‚Üì
[Validate SQL Safety]
    ‚Üì
[Execute on Database]
    ‚Üì
[Format Results]
    ‚Üì
User Response
```
